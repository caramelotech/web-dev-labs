# Projetos

Projetos práticos para consolidar o aprendizado de desenvolvimento web.

## Como usar

1. Leia o objetivo e os requisitos antes de começar
2. Construa o projeto do zero, sem usar frameworks
3. Compare sua solução com a dos colegas ao final

## Projeto 01 - Landing Page Pessoal

**Objetivo:** Construir uma landing page completa usando HTML e CSS puro.

**Contexto:**
Você foi contratado para criar uma landing page de apresentação pessoal ou de um produto imaginário. A página deve ser visualmente agradável e funcionar bem em diferentes tamanhos de tela.

**Requisitos:**

- [ ] Página única com as seções: Hero, Sobre, Habilidades/Funcionalidades, Contato
- [ ] Navegação fixa no topo com scroll suave para as seções
- [ ] Layout responsivo (funciona em mobile e desktop)
- [ ] Usa apenas HTML e CSS (sem JavaScript obrigatório)
- [ ] Carrega sem erros no navegador

**Entregável esperado:**

Um arquivo `index.html` e um `style.css` com uma landing page funcional. A página deve ter aparência profissional e uso correto de tags semânticas.
