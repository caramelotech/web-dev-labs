# Web Dev Labs

Base de conhecimento sobre **desenvolvimento web** do Caramelo Tech - do HTML ao system design.

## O que você vai encontrar

- **Fundamentos Web** - HTML e CSS
- **Backend** - SQL, HTTP, APIs e REST
- **Engenharia de Software** - princípios SOLID
- **System Design** - consistência transacional, encurtador de URL e dual-write problem
- **Ferramentas** - Git, Conventional Commits, Pull Requests, fluxos de trabalho e assinatura GPG

Navegue pelas seções na barra lateral. Se está começando, comece por Fundamentos Web.

## Além das notas

O repositório [web-dev-labs](https://github.com/caramelotech/web-dev-labs) também contém exercícios e projetos práticos na pasta `examples/`.
