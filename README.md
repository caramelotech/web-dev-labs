# Desenvolvimento Web para Iniciantes

Bem-vindo ao repositório de conhecimento sobre desenvolvimento web para iniciantes! Este repositório contém uma base de conhecimento organizada em tópicos sequenciais para facilitar o aprendizado.

## 📚 Estrutura do Conteúdo

Os tópicos estão organizados em pastas numeradas para indicar a sequência recomendada de aprendizado:

1. **[Fundamentos da Programação Web](1-fundamentos-da-programacao-web/)** - Conceitos básicos e introdução
2. **[HTML](2-html/)** - Estrutura e marcação de páginas web
3. **[CSS](3-css/)** - Estilização e layout de páginas web
4. **[JavaScript](4-javascript/)** - Programação e interatividade

## 🎯 Como Usar Este Repositório

1. Navegue pelas pastas na ordem numérica indicada
2. Cada pasta contém um README.md com índice dos conteúdos disponíveis
3. Leia os arquivos .md dentro de cada pasta para aprender sobre cada tópico

## 🤝 Contribuindo

Sinta-se à vontade para contribuir com novos conteúdos ou melhorias! Certifique-se de:
- Adicionar novos arquivos .md nas pastas apropriadas
- Atualizar o README.md da pasta com o link para o novo conteúdo
- Manter a linguagem simples e acessível para iniciantes
