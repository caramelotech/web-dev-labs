# Projetos

Projetos praticos para consolidar o aprendizado de desenvolvimento web.

## Como usar

1. Leia o objetivo e os requisitos antes de comecar
2. Construa o projeto do zero, sem depender de frameworks
3. Compare sua solucao com a de outras pessoas no fim do processo

## Projeto 01 - Landing Page Pessoal

**Objetivo:** Construir uma landing page completa usando HTML e CSS puro.

**Contexto:**
Voce foi contratado para criar uma landing page de apresentacao pessoal ou de um produto imaginario. A pagina deve ser visualmente agradavel e funcionar bem em diferentes tamanhos de tela.

**Requisitos:**

- [ ] Pagina unica com secoes de hero, sobre, habilidades ou funcionalidades e contato
- [ ] Navegacao fixa no topo com scroll suave
- [ ] Layout responsivo para mobile e desktop
- [ ] Usa apenas HTML e CSS
- [ ] Carrega sem erros no navegador

**Entregavel esperado:**

Um arquivo `index.html` e um `style.css` com uma landing page funcional, uso correto de tags semanticas e boa hierarquia visual.
