# Exercicios

Pratique o que aprendeu nas anotacoes e exemplos.

## Como usar

1. Leia o enunciado do exercicio
2. Tente resolver por conta propria antes de buscar ajuda
3. Consulte `src/content/docs/` se precisar revisar algum conceito

## Exercicio 01 - Estruturando uma Pagina HTML

**Objetivo:** Criar a estrutura semantica de uma pagina web simples.

**Instrucao:**
Crie um arquivo `index.html` com a estrutura de uma pagina pessoal contendo:

1. Um `<header>` com seu nome e uma `<nav>` com links de navegacao
2. Um `<main>` com uma secao `<section>` de apresentacao e um `<article>` com um texto sobre voce
3. Um `<aside>` com uma lista de habilidades
4. Um `<footer>` com informacoes de contato

**Criterios de sucesso:**

- [ ] Usa as tags semanticas corretas para cada parte
- [ ] O HTML e valido
- [ ] A pagina abre corretamente no navegador
- [ ] Nao usa `<div>` onde uma tag semantica seria mais adequada

## Exercicio 02 - Estilizando com CSS

**Objetivo:** Aplicar estilos ao HTML criado no exercicio anterior usando seletores CSS.

**Instrucao:**
Crie um arquivo `style.css` e aplique estilos a pagina do exercicio 01:

1. Defina uma paleta de cores usando variaveis CSS
2. Estilize a `<nav>` para exibir os links horizontalmente
3. Aplique um efeito de `:hover` nos links da navegacao
4. Use Flexbox ou Grid para organizar o layout principal

**Criterios de sucesso:**

- [ ] Usa variaveis CSS para as cores
- [ ] A navegacao esta horizontal com efeito hover
- [ ] O layout usa Flexbox ou Grid
- [ ] A pagina e legivel em telas menores
