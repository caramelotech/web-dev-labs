## Exercício 01 - Estruturando uma Página HTML

**Objetivo:** Criar a estrutura semântica de uma página web simples.

**Instrução:**
Crie um arquivo `index.html` com a estrutura de uma página pessoal contendo:

1. Um `<header>` com seu nome e uma `<nav>` com links de navegação
2. Um `<main>` com uma seção `<section>` de apresentação e um `<article>` com um texto sobre você
3. Um `<aside>` com uma lista de habilidades
4. Um `<footer>` com informações de contato

**Critérios de sucesso:**

- [ ] Usa as tags semânticas corretas para cada parte
- [ ] O HTML é válido (sem tags mal fechadas ou aninhamento incorreto)
- [ ] A página abre corretamente no navegador
- [ ] Não usa `<div>` onde uma tag semântica seria mais adequada

## Exercício 02 - Estilizando com CSS

**Objetivo:** Aplicar estilos ao HTML criado no exercício anterior usando seletores CSS.

**Instrução:**
Crie um arquivo `style.css` e aplique estilos à página do exercício 01:

1. Defina uma paleta de cores usando variáveis CSS (`--cor-primaria`, `--cor-fundo`, etc.)
2. Estilize a `<nav>` para exibir os links horizontalmente
3. Aplique um efeito de `:hover` nos links da navegação
4. Use Flexbox ou Grid para organizar o layout principal

**Critérios de sucesso:**

- [ ] Usa variáveis CSS para as cores
- [ ] A navegação está horizontal com efeito hover
- [ ] O layout usa Flexbox ou Grid (sem float)
- [ ] A página é legível em telas menores (responsividade básica)
